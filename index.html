Projects

  • Tabloid

    Built with C# this application, conceptually similar to the blog site Medium, was built and rebuilt as a console application, a web application using MVC, and as a full stack application using React. My contribution to this was implementing CRUD functionality for journal entries feature with console application, CRUD functionality for category feature the MVC version, and CRUD functionality for comments feature for the full stack version.

  • Nutshell

    Nutshell is a dashboard for people to use to organize their daily tasks, events, news articles, friends, and chat messages. In this Group Project, we utilized React and JSON database. My contribution to this was implementing read functionality for messages which involved integrating data from the user provider, and all group efforts.

  • Holiday Road

    This vacation planning application was built using peer and individual programing in the plan, coding, and implementation phases. Features included sorting features to plan a trip location, food options, monuments, and weather of the day that used an external API. My contributions were the attraction feature, and all group efforts.

  • Tender Tofu

    Tender Tofu takes the guess work out of deciding which vegetarian cuisine to enjoy in Nashville. By presenting the numerous restaurant choices for both users/ friends to chose from, Tender Tofu presents the restaurant favorite they have in common for the particular outing. This app was Built with React, CSS, short polling, and the Yelp API.

  • Before the Pen

    Before the Pen is a CRUD web application, built with React, C#, and SQL, that keeps track of bullet journal layouts. Bullet Journaling is a highly customizable organizational system that encompasses the cross-section between a journal, planner, to-do list, and sketchbook. Before the Pen provides a location to save bullet journal resources from across the web, plan which layouts are needed in the upcoming month, and provide data about which layouts were used in the past months.
